# Description

A command-line to-do list application that enables users to manage tasks through a text-based interface.

## Features
- Add tasks
- List tasks
- Mark tasks as complete
- Delete tasks

## Purpose
The project demonstrates core programming concepts such as command-line input handling, persistent storage, and basic data structure usage. 

## How to run
Clone the repository:
```bash
git clone https://github.com/c0dingwkurtley-r/ToDo-Today
```
Navigate to the project directory:
```bash
cd ToDo-Today
```
Compile the program:
```bash
javac *.java
```
Run the program:
```bash
java ToDoList
```

## Technologies Used
- Java
- File I/O for persistent storage
